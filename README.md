# Chess Game in Local host

# Steps to play
  1. Clone The code
  2. Open Terminal and run npx nodemon
  3. Go to any browser and type localhost server (ex: (http://localhost:3080/))
  4. Now run localhost several time
  5. Now play the game for both side server
